## 🗺️ Fluxos Principais

### 📱 User Flows (6 fluxos)

1. **Onboarding** (4 passos)
   - Login → Verificação → Perfil → Boas-vindas

2. **Compra** (7 passos)
   - Home → Produto → Chat → Negociação → Pagamento → Confirmação → Avaliação

3. **Venda** (8 passos)
   - Home → Criar P1 → Criar P2 → Publicado → Chat → Negociação → Venda → Avaliação

4. **Comunicação**
   - Inbox → Conversa → Mensagens

5. **Lista de Desejos**
   - Feed → Favoritar → Lista → Produto

6. **Gerenciamento**
   - Meus Anúncios → Ativos/Vendidos → Editar/Deletar
